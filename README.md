# 视频处理工具包

## 项目简介
本项目是基于Python和OpenCV开发的视频流处理系统，主要用于计算机视觉学习和智能网联汽车视觉处理实践。系统提供了从视频采集、处理到智能分析的全套解决方案，特别适合用于教育场景和智能网联汽车视觉处理实践。通过本系统，用户可以快速掌握计算机视觉的基本原理，并将其应用于实际项目中。

## 核心功能

1. 视频流采集
- 支持笔记本内置/外接摄像头
- 多摄像头选择

2. 实时展示
- GUI/OpenCV窗口显示
- 支持窗口缩放拖动

3. 视频录制
- 定时存储（可设间隔）
- 支持MP4/AVI格式
- 手动控制录制

4. 视频处理
- 分辨率调整
- 帧率调整
- 格式转换

5. 视频回放
- 内置播放器
- 支持快进/暂停

6. 智能分析
- 人脸/车辆/行人检测
- 支持实时与离线分析

## 安装说明
### 系统要求
- 操作系统：Windows 10/11
- Python版本：3.12
- 推荐拥有NVIDIA GPU的电脑，且CUDA版本高于11.8
- 内存：建议4GB以上

### 安装步骤
1. 安装依赖：
```bash
pip install -r requirements.txt
```

## 使用方法
1. 运行主程序：
```bash
python main.py
```
2. 在GUI界面中选择需要的功能模块

### 主界面功能说明
主界面包含以下功能按钮和控件：
- **摄像头选择下拉框**：选择要使用的摄像头设备
- **打开摄像头**：打开摄像头预览窗口，可进行实时视频预览和检测
- **视频录制设置**：设置视频录制参数，如格式、分辨率、帧率等
- **本地视频格式处理**：对本地视频进行格式转换、分辨率调整等处理
- **本地视频回放**：播放本地视频文件，支持快进/暂停等操作
- **视频智能分析设置**：设置视频分析参数，如人脸检测、车辆检测等

### 摄像头预览窗口功能
- **开始/停止录制**：控制视频录制
- **检测选项**：勾选需要进行的检测类型（人脸、行人、车辆）
- **实时预览**：显示摄像头捕捉的画面

3. 按照界面提示进行操作

